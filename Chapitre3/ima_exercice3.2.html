Exercice 3.2 : Utilisation de la pipette de Firefox

Vous allez utiliser la pipette de Firefox pour récupérer le code couleur des quatre carrés ci-dessous. Pour ouvrir cette pipette il faut :

  • dans Firefox cliquer sur Ouvrir le menu (les 3 barres horizontales ≡ en haut à droite sous la croix rouge) ;
  • puis choisir Développement web (Web Developer en Anglais) ;
  • choisir ensuite Pipette (Eyedropper en Anglais) ;
  • et enfin déplacer la souris sur les quatre carrés ci-dessous afin de le code couleur.


Utiliser cette pipette pour remplir les zones ci-dessous, avec les codes RVB des couleurs des carrés.
