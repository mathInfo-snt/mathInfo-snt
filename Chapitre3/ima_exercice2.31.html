Exercice 2.3 : Dématriçage bilinéaire

Premier exemple

On considère la matrice de photosites ci-dessous, avec le filtre de Bayer. Le code RVB du pixel correspondant au photosite central est le suivant :

RVB(50,120,80)


Cliquer sur les photosites une ou plusieurs fois pour changer leur couleur afin qu'elles correspondent au code RVB ci-dessus.
