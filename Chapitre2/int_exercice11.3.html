Exercice 11.3 : Chemin parcouru par des données numériques

On considère le lien suivant Tracer une route internet qui permet de visualiser le parcourt des données numériques lors d'une requête http.

  • Cliquer sur le lien ci-dessus, puis cliquer sur le bouton jaune Run another traceroute.
  • Dans Source choisir Buenos Aires en Argentine.
  • Dans Site/IP choisir le site www.ac-grenoble.fr.
  • Cliquer ensuite sur le bouton jaune situé en dessous.
  • Sur votre cours compléter les zones en pointillés (Exercice 11.3 sur la feuille 11) en donnant les différentes étapes du parcourt des données numériques et la distance totale parcourue par ces données pour cette requête.
