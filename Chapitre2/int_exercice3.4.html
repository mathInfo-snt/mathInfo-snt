Exercice 3.4 : Connecter un nouvel ordinateur à un sous réseau

Connecter un nouvel ordinateur à un sous réseau

Dans /élève/nom.prenom/Mesdocuments/ créer un nouveau dossier nommé Filius.
Lancer le logiciel Filius que l'on trouve dans /Appli/ et ouvrir le fichier nommé reseau1.fls qui se trouve dans le dossier Ressources
dont le chemin est : /Echanges/Votre_Classe/SNT/COURS/Ressources/

L'image du réseau suivant s'affiche dans le logiciel Filius.


  • Sauvegarder ce fichier dans votre dossier Filius en cliquant sur la disquette noire située en haut à gauche dans la barre d'outils.
  • Avec la souris faire glisser un nouveau portable depuis le panneau latéral gauche.
  • Double cliquer sur ce portable, puis dans la zone qui s'affiche en bas, modifier le nom de ce portable en l'appelant M7 et lui attribuer l'adresse IP 192.168.1.4 pour le raccorder au premier sous-réseau situé à gauche (switch R1).
  • Connecter ce portable en cliquant à gauche sur le câble rouge, puis en cliquant d'abord sur le switch R1 puis sur le portable M7.
  • Cliquer sur un autre équipement pour déselectionner le cable.
  • Sur votre cours papier compléter les pointillés de l' Exercice 3.4 sur la Feuille 3.
  • Pour vérifier que cet ordinateur est connecté, il faut lancer une simulation en cliquant sur la flèche verte située dans la barre d'outils en haut.
  • Ensuite, il faut cliquer sur l'ordinateur M1 puis sur la ligne de commande.
  • Dans la console qui s'affiche, taper la commande : ping 192.168.1.4 et vérifier qu'on obtient bien la réponse :
    4 paquets transmis, 4 paquets reçus, 0% paquets perdus.
  • Lever la main pour montrer que les 4 paquets ont bien été reçus.

Point 3 :
Lever la main pour que le professeur vérifie que tous les exercices ont été faits (liste complètement verte à gauche) et que la feuille du cours a été complétée correctement.
