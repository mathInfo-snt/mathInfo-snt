Les couches du protocole TCP/IP

Pour transmettre des données numériques sur le réseau Internet on a vu qu'il fallait rajouter des informations successives : c'est le principe de l'encapsulation. Pour le protocole TCP/IP il y a quatre couches d'informations que l'on va rajouter successivement aux données de départ. Ces couches sont dans l'ordre :

  • la couche Application qui utilise le protocole HTTP pour les pages Web
  • la couche Transport qui utilise le protocole TCP ou UDP
  • la couche Internet qui utilise le protocole IP
  • la couche Lien (ou couche accès réseau) qui utilise le protocole ethernet
