Exercice 1.1 : Adresse IP de l'ordinateur que vous utilisez

Sur votre cours (Exercice 1.1 de la Feuille 1), compléter les pointillés par les adresses IP locale et publique de votre ordinateur.

  • Pour obtenir l'adresse IP locale, il suffit de regarder en bas à droite sur l'écran du bureau, après avoir réduit les applications en fonctionnement.
  • Pour obtenir l'adresse IP publique, il suffit de cliquer sur le lien suivant : adresse IP publique.

Point 1 :
Lever la main pour que le professeur vérifie que tous les exercices ont été faits (liste complètement verte à gauche) et que la feuille du cours a été complétée correctement.
