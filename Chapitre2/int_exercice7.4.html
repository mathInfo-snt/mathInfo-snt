Question 1

Dans cette question, on suppose que le routeur C est en panne, sans se soucier des tables de routages, indiquer un parcours possible pour aller de la machine T1 à la machine A1 sans passer par le routeur C. Noter ce parcours possible sur votre cours (Exercice 7.4 sur la feuille 7).

Question 2

Dans cette question, on suppose que seul le câble entre le routeur A et le routeur C ne fonctionne pas. Sur votre cours (Exercice 7.4 sur la feuille 7) réécrire la table de routage du routeur A et celle du routeur C pour que l'information puisse circuler entre toutes les machines.
