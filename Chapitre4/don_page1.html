Généralités sur les données

Données

Une donnée est la représentation d’une information. Bien avant la naissance de l’informatique, on a eu besoin de stocker, puis de structurer ces informations, pour pouvoir les utiliser. Les données constituent désormais la matière première de toute activité numérique. À ce titre, de nouvelles formes de stockage, de structuration et d’exploitation ont vu le jour au regard de l’explosion de la quantité de données disponibles, jusqu’à l’émergence d’une science du traitement des données : la data science. Les données sont essentielles pour toute activité numérique (gestion de stock, fiche client…).

• Définition (données) : Une donnée est un élément (chaîne de caractères, nombre, image…) décrivant un objet (individu, société, événement, machine, fichier…). Par exemple, le numéro de téléphone d’un contact est une donnée.

Data science et données personnelles

• Définition (data science) : La data science est la science du traitement des données.

• Définition (données personnelles) : Une donnée personnelle est une information se rapportant à une personne physique identifiée ou identifiable par référence à des éléments qui lui sont propres (nom, numéro de sécurité sociale…).



Structuration des données

Structurer correctement des données permet de les utiliser et exploiter aisément afin de produire de l'information. Il faut donc organiser et classer les données.

Objets, descripteurs et valeurs

Définition (objet) : Un objet est un des individus (éléments) sur lequel on collecte des données.

Définition (descripteur) : Un descripteur permet de préciser la signification des valeurs caractérisant un objet.

Définition (valeur) : Une valeur est une des données caractérisant un objet.

Plusieurs descripteurs peuvent être utiles pour décrire un même objet.
Par exemple, un passeport français contient plusieurs descripteurs caractérisant un objet qui ici est une personne :

  • le numéro du passeport,
  • le nom,
  • les prénoms,
  • le sexe,
  • la taille,
  • la couleur des yeux,
  • la date de naissance,
  • le lieu de naissance...
En particulier, sur le passeport de l'acteur français Pierre Richard :
  • à côté du descripteur : lieu de naissance, il est inscrit la valeur : Valenciennes ;
  • à côté du descripteur : date de naissance, il est inscrit la valeur : 16/08/1934.

Collection de données

Définition (Collection de données) : Une collection de données est un moyen de regrouper de manière structurée des objets partageant les mêmes descripteurs.

Elle est généralement représentée sous la forme d'un tableau (appelé aussi table) avec : les descripteurs en tête de colonne, les objets en ligne et les valeurs dans les cellules de ce tableau. Voici une collection de données contenant deux objets. Quatre descripteurs sont utilisés et huit valeurs sont visibles.

Nom Prénom Date de naissance Lieu de naissance
Macron Emmanuel 21 12 1977 Amiens
Hollande François 12 08 1954 Rouen

Ces tableaux peuvent être stockés dans un fichier avec un format spécifique comme le format CSV (Comma-Separated Values). Dans un tel fichier, les données sont sous un format texte simple et séparées les unes des autres par un caractère appelé séparateur (souvent la virgule). Chaque ligne correspond à une ligne du tableau. On peut ouvrir un fichier CSV avec un éditeur de texte basique comme Bloc-notes ou Notepad++ par exemple. Voici ce qu'on obtient :

 Nom,Prénom,Date de naissance,Lieu de naissance
 Macron,Emmanuel,21 12 1977,Amiens
 Hollande,François,12 08 1954,Rouen


Base de données

Définition (Base de données) : Une base de données (database en Anglais) est un ensemble d'informations structurées accessibles au moyen d'un logiciel.

Une base de données permet de stocker et de retrouver l'intégralité de données brutes ou d'information en rapport avec un thème ou une activité.
Le système de gestion de base de données (SGBD) est une suite de programmes qui manipule la structure de la base de données et dirige l'accès aux données qui y sont stockées.

Traitement des données structurées

Avec un tableur : On peut ouvrir un fichier CSV avec un tableur (Excel ou libre office) qui devrait (après lui avoir indiqué que le séparateur utilisé est la virgule par exemple) utiliser une colonne par descripteur et une ligne par objet.
On peut alors avec le tableur, traiter ces données, par exemple on peut les trier, les filtrer, les représenter sous forme de diagramme,...

Avec un langage de programmation : Le langage Python est un langage adapté au traitement des données.



Stockage et métadonnées

Stockage : Les fichiers de données sont stockés sur des supports de trois types différents :

  • internes (SSD (Solid State Drive) ou disque dur);
  • externes (disque, clé USB);
  • distants (cloud).
Aujourd’hui, des centres de données (Data centers) hébergent un nombre toujours croissants de données mais posent des problèmes écologiques : consommation d’énergie en hausse pour fonctionnement des serveurs et des climatisations nécessaires.

Métadonnées : À tout fichier sont associées des métadonnées qui permettent d’en décrire le contenu. Ces métadonnées varient selon le type de fichier (date et coordonnées de géolocalisation d’une photographie, auteur et titre d’un fichier texte, etc.)



Repères historiques

  • 1930 :

    Utilisation des cartes perforées, premier support de stockage de données numériques.

  • 1956 :

    Invention du disque dur permettant de stocker de plus grandes quantités de données, avec un accès de plus en plus rapide.

  • 1970 :

    Invention du modèle relationnel (E. L. Codd) pour la structuration et l’indexation des bases de données.

  • 1979 :

    Création du premier tableur, VisiCalc.

  • 2000 :

    Emergence de la data science.

  • 2009 :

    Open Government Initiative, porté par le président Barak Obama.

  • 2013 :

    Charte du G8 pour l’ouverture des données publiques.
