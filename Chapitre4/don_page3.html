Formats de fichiers

Pour assurer la persistance des données, ces dernières sont stockées dans des fichiers.
Il y a deux grands types de formats de fichiers.

Le type texte

Le type texte regroupe les formats de fichiers lisibles par des éditeurs de texte comme Notepad++, Bloc-Notes, Geany, etc...
Ces fichiers ne contiennent que des caractères alphanumériques qui sont codés en respectant une norme d'encodage, telle que l'utf-8 par exemple, qui est compatible avec le code ASCII.

Voici quelques exemples importants de formats de type texte :

  • le format CSV (Comma Separated Values) qui signifie données avec des séparateurs (virgule par exemple). C'est le format le plus universel et le plus simple.
    Ces fichiers possèdent l'extension .csv.
  • le format VCARD (Visit CARD) qui signifie carte de visite. Ce format est utilisé par des appareils mobiles (téléphones mobiles, assistants personnels, GPS) pour échanger des éléments de carnet d'adresses, mais aussi par des logiciels de messagerie instantanée.
    Ces fichiers possèdent l'extension .vcf.
  • le format XML (eXtensible Markup Language) qui signifie un langage de balisage extensible. Ce format de fichier est conçu pour transmettre des informations. Il utilise des balises comme dans le langage HTML.
    Ces fichiers possèdent l'extension .xml.
  • le format JSON (JavaScript Object Notation) qui signifie notation objet issue de JavaScript. C'est un format léger d'échange de données. JSON est un format texte qui utilise des conventions familières à tout programmeur habitué aux langages descendant du C, comme par exemple : C++, Java, JavaScript, Python et bien d'autres.
    Ces fichiers possèdent l'extension .json.

Le type binaire

Ces fichiers ne sont pas lisibles par les éditeurs de texte. Ce sont principalement les formats des tableurs.

  • le format ODS (OpenDocument Spreadsheet) qui est le format du tableur LibreOffice Calc.
    Ces fichiers possèdent l'extension .ods.
  • le format XLS (EXceL Sheet) qui est le format du tableur Excel.
    Ces fichiers possèdent l'extension .xls.
Il y a aussi les formats utilisés par les logiciels de bases de données :
  • le format DBF qui est le format utilisé par le logiciel de système de gestion de base de données dBASE pour le stockage des données.
    Ces fichiers possèdent l'extension .dbf.
  • le format MYD qui est le format utilisé par le logiciel de système de gestion de base de données MySQL pour le stockage des données.
    Ces fichiers possèdent l'extension .myd.
