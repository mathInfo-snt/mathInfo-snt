Calculs de résumés numériques avec LibreOffice Calc

Fonctions :

Le tableau ci-dessous donne les différentes fonctions de LibreOffice permettant de calculer les résumés numériques d'une série de données.

Action Fonction Syntaxe
Calculer une moyenne MOYENNE() =MOYENNE(plage_de_cellules)
Calculer une médiane MEDIANE() =MEDIANE(plage_de_cellules)
Calculer un quartile QUARTILE() =QUARTILE(plage_de_cellules;numéro_du_quartile)
Calculer un maximum MAX() =MAX(plage_de_cellules)
Calculer un minimum MIN() =MIN(plage_de_cellules)

Plage de cellules :

Une plage de cellules est en général :

  • une plage de cellules réparties dans une colonnes et se note dans ce cas : "première_cellule : dernière_cellule", par exemple : B6:B10
  • une plage de cellules réparties dans une ligne et se note dans ce cas : "première_cellule : dernière_cellule", par exemple : B2:M2
  • une plage de cellules réparties dans un tableau et se note dans ce cas : "cellule_en_haut_à_gauche : cellule_en_bas_à_droite", par exemple : B2:M10

Tri avec LibreOffice Calc

Procédure :

Pour effectuer un tri sous LibreOffice Calc, il faut :

  • sélectionner la plage de cellules que l'on veut trier ;
  • cliquer ensuite sur l'onglet Données ;
  • choisir Tri croissant ou Tri décroissant suivant le tri que l'on veut obtenir ;
  • si demandé, il faut ensuite cliquer sur Sélection active si on ne veut trier que les cellules sélectionnées ;
