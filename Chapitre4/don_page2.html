Introduction

Le format VCARD (Visit CARD) est un format standard ouvert d'échange de données personnelles qui veut dire carte de visite.
Ce format est utilisé par le logiciel Carnet d'adresses d'Apple ou encore les logiciels de courrier électronique comme Mozilla Thunderbird ou Outlook Express.
Le format VCARD est également utilisé par des appareils mobiles (téléphones mobiles, assistants personnels, GPS) pour échanger des éléments de carnet d'adresses, mais aussi par des logiciels de messagerie instantanée comme Jabber ou Skype.
Le fichier VCARD est un format de carnet d'adresses qui peut contenir une ou plusieurs adresses.

Un fichier VCARD possède une extension .vcf (acronyme de Virtual Card File). Son contenu est lisible par n'importe quel éditeur de texte (Bloc Note, Notepad++, ...). Il est utile pour conserver ou donner ses coordonnées personnelles ou professionnelles.


Exemple de fichier VCARD

On considère le fichier Paul_Legrand.vcf suivant :

 BEGIN:VCARD
 VERSION:2.1
 N:Legrand;Paul
 FN:Paul Legrand
 BDAY:19850527
 ADR:12;Rue;Servan;75011;Paris;France
 TEL;CELL:060-735-1493
 TEL;HOME:091-515-6570
 EMAIL:paul.legrand@laposte.net
 END:VCARD


Description détaillé d'un fichier VCARD

Un fichier VCARD doit commencer par la ligne BEGIN:VCARD et se terminer par la ligne END:VCARD.
Le tableau ci-dessous présente les principaux descripteurs du format VCARD.
Les trois premiers descripteurs en bleu sont obligatoires les autres sont optionnels.


Nom du descripteur Description Signification Propriétés
VERSION Version Version de format VCARD Un seul champ texte.
Il existe 3 versions : 2.1 ; 3.0 et 4.0
N Nom Une représentation structurée du nom de la personne ou de l'objet associé à la VCARD Plusieurs champs textes séparés par des points virgules :
Nom de Famille;Prénom(s);Titre(s);...
FN Nom formaté Une chaîne formatée représentant le nom associé à la VCARD Un seul champ texte
BDAY Date de naissance Date de naissance de l'individu associé à la VCARD Un seul champ texte au format :
AAAAMMJJ
ADR Adresse Une représentation structurée de l'adresse de l'individu associé à la VCARD Plusieurs champs textes séparés par des points virgules :
Numéro de rue;Type de rue;Nom de rue;Code postal;Ville;Pays
TEL;CELL Téléphone portable Le numéro de téléphone portable de l'individu associé à la VCARD Un seul champ texte au format :
nnn-nnn-nnnn
TEL;HOME Téléphone fixe Le numéro de téléphone fixe de l'individu associé à la VCARD Un seul champ texte au format :
nnn-nnn-nnnn
EMAIL Adresse mail L'adresse mail de l'individu associé à la VCARD Un seul champ texte

Il existe d'autres descipteurs, vous trouverez une liste plus complète ici .
