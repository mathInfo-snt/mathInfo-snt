Exercice 3.2 : écrire un nom de domaine

Sur votre cours (Exercice 3.2 de la feuille 3)), écrire le nom de domaine puis l'hôte d'un site sachant que :

  • le sous-domaine est www
  • le domaine de premier niveau est ch
  • le domaine de second niveau est admin
  • le domaine de troisième niveau est bakom
Point 3 :
Lever la main pour que le professeur vérifie que tous les exercices ont été faits (liste complètement verte à gauche) et que la feuille du cours a été complétée correctement.
