Introduction

Le Web désigne le World Wide Web, c'est à dire litéralement "la toile d'araignée à l'échelle mondiale". Le Web est une application d'internet qui a été développée au CERN (Conseil Européen pour la Recherche Nucléaire) par le britanique Sir Timothy John Berners-Lee et le belge Robert Cailliau au début des années 90. Tim Berners-Lee met au point le système hypertexte qui permet, à partir d'un document, de consulter d'autres documents en cliquant sur des mots clés. Ces mots "cliquables" sont appelés hyperliens (ou liens) et sont souvent en bleu, comme dans l'exemple ci-dessous (une page web wikipédia).



Quelques définitions

Définition (Hyperlien) : Un hyperlien est un mot clé cliquable permettant de passer automatiquement d'un document consulté à un autre document, notamment d'une page web vers une autre page web.

Définition (Navigateur) : Un navigateur est un logiciel qui permet d'afficher une page web.
Exemples : Mozilla Firefox, Google Chrome, Internet Explorer, Safari.

Définition (Moteur de recherche) : Un moteur de recherche est une application qui permet de trouver des pages web pour les afficher dans un navigateur, en rapport avec des mots clés saisis par l'utilisateur. Pour cela, les moteurs de recherche indexent les pages web par rapport aux mots qu'elles contiennent.
Exemples : Google, Qwant, DuckDuckGo, Bing, Yahoo.

Définition (FAI) : Un FAI (Fournisseur d'Accès à Internet) est un organisme (généralement une entreprise mais parfois aussi une association) offrant une connexion à Internet (le réseau informatique mondial).
Exemples : Orange, Free, SFR, Bouygues.

Définition (Client) : le client désigne l'ordinateur de l'utilisateur qui ouvre un navigateur pour afficher des pages Web.

Définition (Serveur) : le serveur désigne l'ordinateur qui héberge les pages Web recherchées par l'utilisateur.
Ces pages Web se trouvent sur un site hébergé sur l'ordinateur serveur.

Définition (Protocole HTTP) : Le protocole HTTP (HyperText Transfert Protocol) est un protocole de communication qui sert à transférer les pages Web du serveur vers le client. Une demande de page Web de la part du client s'appelle une requête HTTP et la réponse du serveur pour transférer la page Web demandée au client s'appelle une réponse HTTP.


Définition (Language HTML) : Le langage HTML (HyperText Markup Language) est un langage de description qui sert à rédiger des pages web.

Définition (Language CSS) : Le langage CSS (Cascading Style Sheet) est un langage complémentaire au langage HTML qui sert à définir le style graphique des pages Web (couleurs, taille de la police, etc..).

Définition (L'adresse URL) : Le l'adresse URL (Uniform Ressource Locator) est un l'adresse qui sert à localiser une page web.


Les bases du Web

Techniquement le web se base sur trois choses :

  • Le protocole HTTP et le protocole HTTPS (HyperText Transfert Protocol Secure) qui est la version sécurisée du protocole HTTP.
  • Les adresses URL qui permettent de localiser une page Web.
  • Le langage de description HTML et le langage CSS qui servent à rédiger des pages Web.
Attention beaucoup de personnes confondent web et internet. Le web est une application d'internet, mais ce sont deux choses différentes. Internet est un réseau qui utilise le protocole IP alors que le web utilise trois technologies : HTTP, URL et HTML. Sur internet, on a d'autre applications les emails par exemple.
Quand on parle du Web, on pense généralement à la partie visible, celle qui est indexée donc accessible par les moteurs de recherche. Pourtant, il existe une grande partie invisible, que l'on appelle le Web profond (Deep Web). Il comprend toutes les pages qui ne sont pas accessibles par les moteurs de recherche classiques. Elles peuvent être de tous les types, mais le plus souvent ce sont des bases de données, des bibliothèques ou des pages protégées par un mot de passe.


Repères historiques

  • 1965 :

    Ted Nelson, sociologue américain, imagine l'accès rapide à des blocs d'informations contenus dans des documents selon l'intérêt du lecteur : le lien hypertexte est né.

  • 1990 :

    Invention du World Wide Web par Tim Berners-Lee au CERN. Le Web est un système hypertexte qui permet de naviguer de documents en documents grâce à des hyperliens. Tim Berners-Lee définit le protocole HTTP, le langage HTML, ainsi que les adresses URL et il crée le premier navigateur Web. La première page HTML mise en ligne est toujours accessible à l'adresse : première page web .

  • 1993 :

    Création du navigateur Mosaic qui va populariser le Web.

  • 1995 :

    Création des langages de programmation Javascript et PHP qui vont permettre de dynamiser les pages Web et les rendre interactives.

  • 2010 :

    Apparition de technologies adaptées qui donnent naissance au Web mobile.
