Construction d'une URL

Définition (URL) : Une URL (Uniform Ressource Locator) est litéralement un localisateur universel de ressources. Il s'agit d'une adresse qui sert à indiquer l'emplacement d'un document web sur un serveur (un ordinateur).

Une URL est constituée de quatre éléments principaux, comme on peut le voir dans l'exemple ci-dessous.


Les quatre éléments de l'URL sont :

  • le protocole (http pour les pages Web, https pour les pages web sécurisées, mailto pour les emails) ;
  • l'hôte (commence souvent par www. suivi du nom de domaine, ici : webdev.com) ;
  • le chemin (il donne l'emplacement de la ressource sur l'ordinateur serveur, ici dans le dossier : /accueil/) ;
  • le fichier (le nom du fichier avec son extension, c'est à dire son type, ici : index.html).


Quelques explications

Hôte et nom de domaine

Il ne faut pas confondre l'hôte et le nom de domaine. L'hôte est en général construit de la façon suivante : sous-domaine.nom-de-domaine.

Sous-domaine

Le sous-domaine www indique habituellement des pages webs standards. C'est le sous-domaine le plus courant.
On peut utiliser d'autres sous-domaines comme :

  • forum
  • video
  • webmail
Cela permet d'ordonner davantage son site. Par exemple on pourrait avoir les hôtes :
  • forum.site-snt.com
  • video.site-snt.com
  • webmail.site.fr
Le nom de sous-domaine n'est pas toujours présent. De plus en plus de site ne l'utilise plus, exemple :
  • http://anna-de-noailles.elycee.rhonealpes.fr

URL sur un site dynamique

Sur un site dynamique (c'est à dire un site qui utilise une base de données) le document recherché est caractérisé par un ensemble de paramètres. Ces paramètres sont indiqués à la fin de l'URL, après le point d'interrogation, comme dans l'exemple ci-dessous.

  • https://www.botanic.com/recherche.html?keywords=acer%20palmatum%20dissectum&_r=true

Nom de fichier

Les documents ou les médias électroniques ont besoin d’un nom de fichier unique pour être identifiés.
Il se compose normalement du vrai nom du fichier suivi d'un point et d'une extension : nom.extension.
L'extension de fichier est indissociable du nom et sert à identifier son format (image, texte, ...).
Voici une liste d'extension courante :

  • .doc : fichier texte word ;
  • .html : fichier pour page web ;
  • .jpg : fichier image au format jpeg ;
  • .mp3 : fichier audio au format mp3 ;
  • .odt : fichier texte libre ofice ;
  • .ods : fichier tableur au format libre ofice ;
  • .pdf : fichier présentation au format pdf ;
  • .png : fichier image au format png ;
  • .txt : fichier texte simple ;
  • .xls : fichier tableur au format excel.
