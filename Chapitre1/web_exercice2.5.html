Exercice 2.5 : représenter graphiquement l'arborescence d'un site web

On considère un site web dont les adresses URL de chacun des fichiers sont les suivantes :

  • http://www.site-snt.fr/main/index.html
  • http://www.site-snt.fr/doc/html/fichier1.html
  • http://www.site-snt.fr/doc/php/fichier2.php
  • http://www.site-snt.fr/doc/css/style.css
  • http://www.site-snt.fr/images/png/im1.png
  • http://www.site-snt.fr/images/jpg/im2.jpg

Sur votre cours (Exercice 2.5 de la feuille 2)), représenter graphiquement l'arborescence de ce site web en partant du serveur et en positionnant ensuite les dossiers et les fichiers avec leurs extensions, comme celui représenté dans l' Exercice 2.2. Mais attention il ne faut pas faire la même arborescence que dans l'exercice 2.2, il faut vous addapter aux six url données ci-dessus.
