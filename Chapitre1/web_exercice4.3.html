Exercice 8 : interpréter certains éléments d'une réponse HTTP

On considère ci-dessous une réponse HTTP d'un serveur :

HTTP/1.1 200 OK
Date: Sat, 23 Mar 2019 13:42:57 GMT Server: Microsoft-IIS/12.0.19288.1
Content-Type : text/html
Content-Length : 1309

Sur votre cours (Exercice 4.3 de la feuille 4), compléter les phrases suivantes :

La date de transfert des données est le .........................................
L'adresse IP du serveur est : .......................................
La longueur en octets de la réponse HTTP est de ................. octets.
