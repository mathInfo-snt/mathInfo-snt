Exercice 6.1 : supprimer des cookies

1. Conserver l'historique :

Pour conserver l'historique :

  • cliquer sur le bouton Ouvrir le menu les trois barres horizontales situées en haut à droite du navigateur ;
  • cliquer ensuite dans le menu déroulant sur Options ;
  • cliquer ensuite dans le menu déroulant à gauche sur Vie privée et sécurité ;
  • dans la rubrique Historique dans Règles de conservation, choisir Conserver l'historique.

2. Enregistrer des données dans deux cookies :

Saisir votre prénom dans la zone ci-dessous, puis cliquer sur le bouton Valider. En cliquant sur ce bouton, cela va enregistrer votre prénom et le nombre de connexion à cette page dans deux cookies différents.


 

3. Afficher les données enregistrées dans les deux cookies :

Les données enregistrées dans ces deux cookies, c'est à dire le prénom saisi et le nombre de connexion à cette page, s'affiche en gras ci-dessous.

  • Quitter la page web de cet exercice, en cliquant sur la croix de l'onglet ou en utilisant les liens du menu de gauche, puis revenez ensuite sur cette page et vous verrez que le prénom reste enregistré dans le cookie et que le nombre de connexion augmente de 1.
  • Cliquer plusieurs fois sur la flèche de rafraîchissement (flèche arrondie située en haut à gauche du navigateur) et vous observerez que le nombre de connexion augmentent de 1 à chaque fois et que votre prénom reste enregistré dans le cookie.

4. Supprimer les deux cookies :

Pour supprimer ces deux cookies, suivre la procédure suivante :

  • cliquer sur le bouton Ouvrir le menu les trois barres horizontales situées en haut à droite du navigateur ;
  • cliquer ensuite dans le menu déroulant sur Options ;
  • cliquer ensuite dans le menu déroulant à gauche sur Vie privée et sécurité ;
  • descendre avec la souris jusqu'à la section Cookies et données de sites ;
  • cliquez sur le bouton Gérer les données… ;
  • dans la liste qui s'affiche, repérer les deux cookies répertoriés par un espace vide dans la première colonne "Site" ;
  • cliquer sur cette ligne (avec un espace vide), puis cliquer en dessous sur le bouton Supprimer les sites sélectionnés, cliquer ensuite sur le bouton Enregistrer les changements ;
  • cliquer alors sur le bouton Supprimer dans la fenêtre qui s'affiche ;
  • revenir à la page de l'exercice 6.1, cliquer sur la flèche de rafraichissement et constater que les données ont bien été supprimées.

5. Valider l'exercice

Rentrer de nouveau votre prénom et cliquer plusieurs fois sur la flèche de rafraichissement pour revalider les deux premières étapes (les deux symboles vert doivent apparaître), puis valider cet exercice en cliquant sur le bouton ci-dessous.
