Exercice 5.1 : exemple d'indexation

Au fur et à mesure qu'il parcourt le Web, le moteur de recherche indexe les contenus.
Il crée un index qui associe à chaque mot l'ensemble des pages qui contiennent ce mot, avec l'importance du mot pour cette page.
Exemple de critère d'importance :
- un mot qui apparaît dans un titre est plus important (5 points) pour cette page que s'il apparaît dans un paragraphe (1 point).
- un mot qui apparaît souvent dans la page est plus important pour cette page que s'il n'apparaît qu'une seule fois (1 point par apparition).


Exemple pour la page : avocat.html

Avocat
En droit l'avocat est un juriste dont les fonctions traditionnelles sont de conseiller, représenter, assister et défendre ses clients en justice, en plaidant pour faire valoir leurs droits.

Index

  mot     importance     explications  
  avocat     6     5+1=6  (1 fois dans le titre et 1 fois dans le paragraphe) 
  droit     2     1+1=2  (2 fois dans le paragraphe)  
  juriste     1     1     (1 fois dans le paragraphe)

Juge
En droit le juge est un magistrat chargé de rendre la justice en appliquant les lois. Le juge est investi officiellement de l'autorité de rendre la justice, un jugement, de dire le droit.
