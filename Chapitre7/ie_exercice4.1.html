Exercice 4.1 : IHM pour un portail

Attendre que le projet se charge...
Voici un exemple d'IHM avec un portail. Cliquer sur le drapeau vert. On peut alors déplacer le camion rouge avec les flèches gauche et droite. Un capteur de présence allume un bouton vert quand le véhicule s'approche du portail. On peut alors ouvrir ce portail en cliquant sur le bouton vert et le camion peut franchir le portail.


L'objectif est de programmer le bouton vert qui sert d'IHM pour ouvrir le portail.
Pour cela, il faut suivre la procédure suivante :
  • Cliquer sur le lien suivant IHM portail.
  • Cliquer sur le bouton en haut à droite Voir à l'intérieur : .
  • Agencer les blocs sélectionnés du sprite Bouton, pour montrer le bouton quand il reçoit le message "présence détectée" et pour qu'il envoie le message "ouvrir" puis cache le bouton quand on clique dessus. L'algorigramme et le pseudo-code de ce programme sont donnés ci-dessous.
  • Une fois le programme agencé (vous devez obtenir trois blocs d'agencement séparés), cliquer sur le sprite Camion, puis cliquer sur le drapeau vert et déplacer le camion avec les flèches du clavier pour vérifier que le programme fonctionne correctement.
