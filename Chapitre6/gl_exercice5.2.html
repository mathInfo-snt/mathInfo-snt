Exercice 5.2 : Faire des calculs dans géoportail

1. On considère le télésiège suivant.




  • Ouvrir le lien suivant : les Brochaux.
  • Cliquer sur la boîte à outils, en haut à droite :
  • Cliquer ensuite sur Mesures.


A. Dans le menu Mesures, choisir Mesurer une distance pour mesurer la distance horizontale du télésiège ci-dessus.
  Faire un simple clic pour le point de départ, et un double clic pour le point d'arrivée.
  Déplacer ensuite le curseur ci-dessous pour qu'il indique cette distance horizontale, à 0,1 km près.



 0.2 km

   


B. Dans le menu Mesures, choisir Etablir un profil altimétrique pour mesurer le dénivelé du télésiège ci-dessus.
  Faire un simple clic pour le point de départ, et un double clic pour le point d'arrivée.
  Déplacer le curseur ci-dessous pour qu'il indique ce dénivelé, à 10 m près.



 20 m

   


2. L'objectif est d'estimer avec Géoportail la surface du lac vert.

  • Toujours avec la même carte : les Brochaux.
  • Zoomer sur le lac vert, à droite du col de Chésery, près de la cote 1981, en Suisse.
  • Cliquer ensuite sur Mesures.


C. Dans le menu Mesures, choisir Mesurer une surface pour mesurer la surface du lac vert.
  Faire un simple clic pour le point de départ et les points intermédiaires, puis un double clic pour le dernier point qui n'est pas le point de départ.
  Déplacer le curseur ci-dessous pour qu'il indique cette surface, à 1000 m² près.



 1000 m²
