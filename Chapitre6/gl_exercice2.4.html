Exercice 2.4 : Se repérer sur le globe terrestre

a. Sur le globe représenté sur votre cours (Exercice 2.4 de la feuille 2) placer les villes suivantes :
  • M pour Montreal ( 45°Nord ; 65°Ouest ) ;
  • R pour Rio ( 25°Sud ; 45°Ouest ) ;
  • V pour la Voulte ( 45°Nord ; 0° ) .
b. Sur votre cours (Exercice 2.4 de la feuille 2) noter les coordonnées GPS des points suivants :
  • O pour Oslo ;
  • I pour Miami ;
  • D pour Saint-Denis.
