Généralités

Un microprocesseur ou CPU (Central Processing Unit) est le coeur de l'ordinateur chargé de traiter et d'exécuter les instructions. On peut simplifier en disant qu'il est chargé d'exécuter les programmes de l'ordinateur.
Ce composant a été inventé en 1971 par Intel (avec le modèle 4004).
Le processeur est rythmé par une horloge (quartz) cadencée plus ou moins rapidement (on parle de fréquence d'horloge mesuré en Hz). A chaque impulsion d'horloge, le processeur lit l'instruction stockée généralement dans un registre d'instruction, une petite mémoire très rapide située dans le processeur, et l'exécute. Dans une même gamme (et donc à architecture comparable) un processeur cadencé plus rapidement est plus efficace, car il peut traiter les instructions plus rapidement.



Architecture de Von Neuman

L’architecture de von Neumann est un modèle structurel d’ordinateur. Les ordinateurs actuels sont tous basés sur des versions améliorées de cette architecture.

L’architecture de von Neumann décompose l’ordinateur en 4 parties distinctes :
  • l’unité arithmétique et logique (UAL ou ALU en anglais) ou unité de traitement : son rôle est d’effectuer les opérations de base ;
  • l’unité de contrôle ou de commande (control unit), chargée du « séquençage » des opérations ;
  • la mémoire qui contient à la fois les données et le programme qui indiquera à l’unité de contrôle quels sont les calculs à faire sur ces données ;
  • les dispositifs d’entrée-sortie, qui permettent de communiquer avec le monde extérieur.


Mémoire

La mémoire dans l'architecture de Von Neuman, correspond à la mémoire vive ou mémoire de RAM. Cette mémoire est volatile c'est à dire qu'elle perd son contenu lorsqu’elle n’est plus alimentée. Elle contient les programmes et les données de ces programmes en cours de fonctionnement.
cette mémoire vive se présente généralement sous la forme de barrettes que l'on clipse sur la carte mère de l'ordinateur.



Attention, la mémoire permanente (disque dur, clé USB, ...) contenant les programmes et données de base de la machine ne fait pas partie de cette composante : la mémoire permanente ne disparait pas à la mise hors tension, elle fait partie de la composante entrées/sorties.


Entrées/sorties

Parmi les entrées/sorties on trouve la mémoire permanente (disque dur interne ou externe, clé USB, etc...), mais aussi tous les périphériques qui permettent d'utiliser un ordinateur (écran, clavier, souris, imprimantes, webcam, etc... ).

Processeur

Le processeur regroupe les deux parties unité de contrôle et unité arithmétique et logique.
l’unité de contrôle permet d’exécuter les instructions (les programmes), elle joue le rôle de « chef d’orchestre » : récupération des instructions et des données pour les envoyer à l’unité arithmétique et logique.
L’unité arithmétique et logique est chargée de l’exécution de tous les calculs que peut réaliser le microprocesseur :

  • opération arithmétiques (sur les nombres entiers ou flottants) ;
  • opérations logiques (sur les bits) ;
Le processeur possède dans ses deux unités, des petites cases mémoires ultra rapides appelées registres. Dans le premier processeur Intel 4004 ces cases mémoires avait une taille de 4 bits. Actuellement la plupart des processeurs ont des registres qui ont une taille de 32 bits ou 64 bits.
L'accumulateur est un exemple de registres qui est utilisé pour stocker les données en cours de traitement par l’unité arithmétique et logique.

Bus de communication

Les flèches représentées dans l'architecture de Von Neuman représentent les bus de communication qui permettent de communiquer les informations (données, instructions, adresses) entre les différentes parties de l'ordinateur.
